# Projet de Classification d'Images avec Keras

Ce projet construit un modèle de classification d'images (ex : MNIST) en utilisant Keras.

## Structure du projet
- `data/` : Dossier pour les datasets
- `notebooks/` : Notebooks pour exploration
- `src/` : Scripts Python (ex : `model.py`)
- `outputs/` : Modèles entraînés et résultats
