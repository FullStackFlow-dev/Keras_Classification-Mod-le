# Keras_Classification-Modèle 

Projet de classification d'images avec Keras, utilisant un modèle séquentiel simple. Inclut la définition, l'entraînement et la sauvegarde du modèle, avec options pour sortie softmax ou logits. Idéal pour MNIST ou autres datasets d'images, avec organisation claire et reprodusable.
# Projet de Classification d'Images avec Keras

Ce projet construit un modèle de classification d'images (ex : MNIST) en utilisant Keras.

## Structure du projet
- `data/` : Dossier pour les datasets
- `notebooks/` : Notebooks pour exploration
- `src/` : Scripts Python (ex : `model.py`)
- `outputs/` : Modèles entraînés et résultats
